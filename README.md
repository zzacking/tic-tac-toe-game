# 🎮 Tic-Tac-Toe Game (Python + OpenCV)

A Tic-Tac-Toe game implemented with Python and OpenCV, featuring both two-player and human-vs-AI modes.

## ✨ Features
- 🆚 Two game modes: P1 vs P2, P1 vs CPU
- 🤖 Smart AI opponent (using Minimax algorithm with Alpha-Beta pruning)
- ↩️ Undo functionality
- 🎯 Automatic win/draw detection
- 🎨 Clean graphical interface with animations
- 📱 Responsive button interactions

## 🚀 How to Run
1. Ensure you have Python installed
2. Install required dependencies:bash -> pip install opencv-python numpy
3. Run the game:bash -> python tictactoe(plus).py

## 🎮 Game Controls
- **P1 vs P2**: Two players take turns
- **P1 vs CPU**: Play against computer AI
- **Undo Button**: Take back the last move(s)
- **Home Button**: Return to main menu
- **Replay Button**: Restart the current game

## 🧠 AI Algorithm
The AI uses the **Minimax algorithm** with **Alpha-Beta pruning** for optimal decision-making in Tic-Tac-Toe.
